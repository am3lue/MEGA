About MEGA

MEGA (Meteorological General Analyst) is a web application that provides weather data analysis. It is built with Genie, a high-performance, high-productivity Julia web framework.

This application uses the Open-Meteo API to fetch weather data.

Data Analysis

Get detailed weather data analysis with charts and tables.

Real-time Data

Fetches real-time weather data from the Open-Meteo API.

Built with Julia

Powered by the high-performance Julia programming language and Genie framework.
